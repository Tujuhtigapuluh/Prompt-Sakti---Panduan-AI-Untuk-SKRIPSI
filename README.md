# 🎓 Kitab Prompt Skripsi untuk AI

Aplikasi web interaktif yang berisi koleksi lengkap **Prompt AI (ChatGPT, Claude, Gemini)** khusus untuk membantu mahasiswa dalam menyusun skripsi. Semua prompt dirancang dengan gaya bahasa akademis, sistematis, logis, dan menggunakan teknik anti-plagiarisme (Turnitin safe).

## ✨ Fitur Utama
- **45+ Prompt Lengkap:** Dari pencarian judul, Bab I hingga Bab V, hingga persiapan sidang.
- **Sistem Role (Prompt Master):** Mengatur "mindset" AI menjadi akademisi ahli sebelum memulai.
- **Pemisahan Metodologi:** Badge khusus untuk prompt Kualitatif (🟢) dan Kuantitatif (🔵).
- **Copy-Paste Instan:** Salin prompt ke clipboard hanya dengan satu klik.
- **Desain Responsif:** Nyaman digunakan di Laptop maupun Smartphone.
- **Fitur Pencarian:** Cari kebutuhan prompt (misal: "Rumusan Masalah") dengan cepat.

## 🚀 Cara Menjalankan Secara Lokal (Local Development)

Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/) di komputer Anda.

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/username/kitab-prompt-skripsi.git
   cd kitab-prompt-skripsi
   ```

2. **Instal dependensi:**
   ```bash
   npm install
   ```
   *Atau jika menggunakan yarn/pnpm:*
   ```bash
   yarn install
   # atau
   pnpm install
   ```

3. **Jalankan server pengembangan (Development Server):**
   ```bash
   npm run dev
   ```

4. Buka browser dan akses alamat `http://localhost:5173`

## 🛠️ Build untuk Produksi (Production Build)

Untuk menghasilkan file statis yang siap di-hosting (seperti di GitHub Pages, Vercel, atau Netlify):

```bash
npm run build
```
File hasil build akan berada di dalam folder `dist/`.

## 🤖 Alur Penggunaan Aplikasi (Best Practice)
1. **Buka bagian "Prompt Master"** di aplikasi.
2. **Salin "Role Setting Ahli Akademik"** dan kirimkan ke AI (ChatGPT/Claude/Gemini). Ini penting untuk memberikan instruksi dasar anti-plagiarisme dan gaya bahasa.
3. **Cari prompt spesifik** yang Anda butuhkan (misal: Identifikasi Masalah).
4. **Isi bagian dalam kurung `(...)`** dengan data asli/konteks skripsi Anda.
5. **Kirim ke AI** dan dapatkan hasil draft dengan bahasa akademis yang sempurna!
